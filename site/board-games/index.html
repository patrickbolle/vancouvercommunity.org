Board Games in Vancouver

4 game night groups plus board game cafes with 600+ game libraries. Drop in, no experience needed.

Know a group? Add it Suggest an edit

Vancouver Board Games #

  • What: Largest regularly meeting board game group downtown
  • Find it: meetup.com/vancouverbg

Board Games Social - WestEnd #

  • What: Relaxed social gaming
  • Find it: meetup.com/board-games-social-westend-vancouver

Good Game - North Vancouver #

  • What: Board gaming meetup with modern games provided. Inclusive, fun
  • Find it: meetup.com/good-game-north-vancouver

Vancouver Social Deduction Gamers #

  • What: Mafia-style games and more
  • Find it: meetup.com/vancouver-mafia-werewolf-group

Venues & Spaces #

Hideout Cafe #

  • What: Biweekly board game meetups
  • Where: 716 Carnarvon Street, New Westminster
  • Cost: Spend $10 on food/drink
  • Find it: thehideoutcafe.com

Mench Café #

  • What: Board gaming & cyber café
  • Where: North Vancouver
  • Find it: menchcafe.com

Pizzeria Ludica #

  • What: Board game café with no play fee. Order food/drinks, play from huge game library
  • Find it: pizzerialudica.com

Storm Crow Alehouse #

  • What: Geek-themed pub with board games available. Nerdy atmosphere, good food and beer
  • Find it: stormcrowalehouse.com

Boardwalk Cafe and Games (Abbotsford) #

  • What: 600+ tabletop games, worth the drive. Massive selection, friendly staff
  • Where: Abbotsford, ~1 hour from Vancouver
  • Find it: abbyboardwalk.com
