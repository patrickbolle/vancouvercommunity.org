  • What: Free social runs and walks, track nights, outdoor adventures
  • Vibe: Community-first, optional training programs
  • Find it: striderz.com

  • What: Free group and community runs
  • Where: Kits
  • Find it: vanrunco.com/pages/flight-crew

    • NEOS Run Club#

  • What: Saturday morning runs
  • Vibe: "More than just a run club — a community"
  • Find it: Meetup / Instagram

    • Vancouver Frontrunners#

  • What: Inclusive running/walking group
  • Vibe: No-drop policy, LGBTQ+ friendly
  • Find it: Search "Vancouver Frontrunners"

    • Wanderung#

  • What: Mailing list for outdoor adventures
  • Size: 3,600+ subscribers, 200+ adventures/year
  • Find it: wanderung.ca

    • Common Ground Run Club#

  • What: Weekly social runs
  • Find it: joincommonground.com

    • Lululemon Run Club#

  • What: Free weekly runs from Lululemon stores
  • Where: Various locations (Robson, Kits, etc.)
  • Vibe: All paces welcome, often includes post-run stretching
  • Find it: Check local Lululemon store schedules
